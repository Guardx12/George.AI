CoachGeorge clean ChatGPT-style rebuild

What changed:
- Replaced the old scripted CoachGeorge component with a conversation-first version.
- Removed shortcut buttons such as "I’m hungry", "Weight went up", and "Bad day".
- Removed dashboard/profile clutter from the main screen.
- Made the George orb the main tap-to-talk control.
- Kept a simple text box underneath for people who prefer typing.
- Added a premium mobile-first dark UI with restrained luxury gold accents.
- Added a new /api/coach-george-chat route so typed and spoken messages go through a proper GPT-style coaching brain rather than keyword/canned replies.
- Added local memory storage for goal, weight, target, tracker, struggles, patterns, food preferences, and short summaries.
- Added natural onboarding logic via conversation instead of forms.
- Added strong system guardrails so George stays focused on food, training, motivation, progress, cravings, routines, and staying on track.
- Voice still uses the existing /api/george-session realtime voice renderer and speaks the exact reply generated by the coaching brain.

Important:
- This version does not add backend user accounts, Supabase, Stripe, notifications, or cross-device memory yet.
- Memory is localStorage-based for the MVP/demo, so it remembers on the same browser/device.
- Server-side chat requires OPENAI_API_KEY to be set in Vercel. If missing, a small fallback coach response still works for basic cases.
