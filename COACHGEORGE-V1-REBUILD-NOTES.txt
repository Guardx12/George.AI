Coach George v1 rebuild

What changed:
- Rebuilt /coachgeorge around a mobile-first voice coach experience.
- Removed the old dashboard-heavy layout, progress percentage, visible setup clutter, and large profile panel from the main view.
- Added a premium dark UI with a blurred gym background, softer champagne/titanium accent, large Coach George avatar/orb, central message card, voice button, chat history, and full-width text input underneath.
- Added a cropped Coach George avatar asset at public/coach-george-avatar.png so the page no longer uses the old blue George logo.
- Removed the browser speechSynthesis fallback that caused the robotic secondary voice.
- Voice output now only comes through the live voice connection, keeping George consistent. Typed messages still show text; when voice is connected, George also speaks the reply.
- Reworked onboarding so George asks naturally and builds the profile behind the scenes instead of forcing a form.
- Strengthened scope control so George stays focused on food, training, motivation, progress, bad days, weight changes, and staying on track.
- Improved memory/pattern handling so repeated signals like hunger, scale anxiety, late-night eating, and motivation dips are saved as coaching patterns rather than full messy transcripts.
- Profile/settings are now hidden behind the settings icon and exist as an edit screen, not the main experience.

Still not included yet:
- Backend user accounts/database.
- Push notifications/proactive reminders.
- Payments/subscriptions.
- Proper analytics.
- A real long-term cross-device memory system.

Important:
This is still localStorage-based for demo/MVP. It will remember on the same device/browser, but backend work is needed before launching properly as a paid app.
