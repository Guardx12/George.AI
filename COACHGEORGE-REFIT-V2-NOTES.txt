CoachGeorge refit v2 - May 14 2026

Main change:
- Removed the forced full-screen setup popup from /coachgeorge.
- Added an inline Quick Coaching Profile card instead.
- Changed onboarding direction so users can either tap to talk or quickly save basic coaching context.

Product direction:
- Coach George is now positioned as a coach beside NutriCheck/MyFitnessPal, not a replacement calorie tracker.
- Exact calorie/gram tracking remains something users do in their tracker.
- George focuses on food decisions, training guidance, motivation, bad-day recovery, weight jumps, hunger, and staying on track.

UI changes:
- The first screen is now built around the Tap to Talk orb.
- New users see a lightweight profile card, not a blocking modal.
- Removed visible age/sex/height setup from the user-facing onboarding.
- The app still uses hidden safe defaults internally until the profile/memory model is rebuilt properly.
- Stats now show placeholder values until a profile is saved.
- Full Reset has been renamed to Reset / Clear Profile.

Still to do next:
- Add proper active phases: Fat loss, Boxing, Strength, Running, Maintenance, General health.
- Add Start New Phase / Edit Profile / Reset Current Phase / Delete All Data as separate actions.
- Replace localStorage with Supabase when moving from demo to real app.
- Add structured coaching memory: goals, struggles, preferences, check-ins, phase history.
- Rewrite the voice/chat intent system so George can collect profile details conversationally and save them into structured state.
- Remove old meal-plan engine code entirely once no longer needed.
